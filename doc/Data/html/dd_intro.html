CHAPTER 1 Introduction to Zebra Data Management

This manual provides information about managing the data side of the Zebra system. Many kinds and formats of data can be integrated with the Zebra data store system, some without any processing whatsoever and others with the application of existing ingest or conversion utilities. For desperate situations, this manual also includes specifications of some of the more common file formats and conventions recognized by Zebra.
