CHAPTER 3 The Location Classes: Location and Boundary

3.1 The Boundary DataChunk Class

The boundary data class (class identifier DCC_Boundary, subclass of DCC_Transparent) is oriented toward the representation of data in the form of outlines or boundaries---essentially a series of locations in physical space. It is a relatively simple class, with just a couple of methods for storing boundaries into data chunks and getting them back out again.

To store a boundary into a data chunk, use dc_BndAdd() as follows: 

void 
dc_BndAdd (dc, when, platform, points, npoint)
DataChunk *dc;
ZebTime *when;
PlatformId platform;
Location *points;
int npoint;

Where dc is a data chunk which is a subclass of DCC_Boundary, when is the time associated with this boundary, platform is the ID of the platform associated with this boundary---this should generally be the same as the platform which appears in the dc_Platform field, points is the series of points which makes up the boundary, and npoint is the number of points in the boundary.

To retrieve a boundary from a data chunk, use the dc_BndGet() function: 

Location *
dc_BndGet (dc, sample, npoint)
DataChunk *dc;
int sample, *npoint;

The return value is a pointer to the series of points corresponding to the given sample index in dc; the number of points will be stored into npoint.

3.2 The Location DataChunk Class

The location class (identifier DCC_Location) is a very simple subclass of DCC_Transparent, useful when the only desired information is a time series of locations. This class may be used, for example, to represent a track of a moving observation platform for which no other data is desired or available.

About the only thing that can be done with this class is to store or fetch locations. Use dc_LocAdd() to add a location to a data chunk: 

void
dc_LocAdd (dc, when, loc)
DataChunk *dc;
ZebTime *when;
Location *loc;

Where dc is a data chunk which is a subclass of DCC_Location, when is the time associated with this location, and loc is the location itself.

To retrieve a location from a data chunk, use dc_LocGet(). 

int dc_LocGet (dc, sample, when, loc)
DataChunk *dc;
int sample;
ZebTime *when;
Location *loc;

Where dc is the data chunk, and sample is the sample of interest. The location and its associated time will be returned in loc and when, respectively.

Since location data chunks are subclasses of DCC_Transparent, you may also use dc_GetLoc() and dc_GetTime() to get the location and time information independently.
