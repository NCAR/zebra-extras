Preface

This manual covers the application interface to the Zebra DataStore library, version 4.2. Much effort has been expended trying to make this document accurate, correct, and complete. However, as nothing can ever be perfect, please report whatever imperfections you discover to granger@ncar.ucar.edu. Also, please contact me at that same address if you need clarifications or details on anything in this manual. Many examples have been added, but we welcome suggestions for additional examples.
