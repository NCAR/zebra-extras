CHAPTER 1 Guide to Documentation

This chapter outlines the manuals and information sources available for Zebra. This is a more complete enumeration than the brief information found in the "Whom to Contact" section of the Installation Manual. Also, a large set of help topics is available through Zebra's on-line help interface, which is introduced in Chapter 3.

1.1 Zebra Manuals

Below are the manuals for the Zebra system, along with a short decription of each.

Release Notes

  • Originally part of the Installation Manual, the release notes are now a separate document for highlighting the the features, changes, and notable notes about each release.

The Zebra Installation Manual

  • This manual provides detailed information about configuring and compiling a Zebra installation.It also includes a list of contacts for problems or questions concerning Zebra.

The Zebra Advanced User's Manual

  • This document describes the user interface of each of the processes in the Zebra system, from the DataStore daemon and its utilities to the graphics process displays and the multitudinous and multifarious plot description parameters. It is a vital reference for anyone trying to enhance, customize, or troubleshoot a Zebra installation.

The Zebra Data Management Guide

  • A guide to assimilating data into the Zebra data store system through ingestor programs, conversion utilities, and write-your-own utilities. It also includes descriptions of supported data formats. This guide is intended for sites which need to incorporate data which is not already included in an RDP dataset or in a Zebra-supported format.

A Guide to Realtime Zebra

  • This is a guide for sites assimilating data in real-time or through automated and event-driven feeds. It provides an overview of the many possible real-time host and networking setups, and includes details on running the Zebra programs available for real-time monitoring, diagnostics, and archiving.

The Zebra Developer's Guide

  • General notes about the Zebra development environment and writing applications using the Zebra libraries. This is the guide to see for information about Zebra's use of the Imake system and about writing and editing Imakefiles for the Zebra development environment.

The Zebra Library Interface Manual

  • This is a programming reference manual describing the application interface utilized by all Zebra clients. The Zebra library interface provides access to the message manager, the timer, sound, primitive graphics, and the configuration variables. Anyone attempting to write any type of customized Zebra client should refer to this manual.

The Zebra DataStore Interface Manual

  • The Zebra DataStore provides the data handling capability of Zebra clients. Zebra clients use the library routines to communicate with the DataStore server process and to read and write data files. This is the manual to consult when writing an ingest or data access module.

Some ancient references to a Configuration Manual may still exist. That manual is now obsolete and has been supplanted by the Advanced User's Manual. Likewise, the DataStore and Library interface manuals are the more complete descendants of the original combined Interface manual.

The Zebra document distribution also includes the Graphics Process Structures and Internals manual. This manual descends from the very original days of Zebra, but its contents are still relevant and useful regarding the implementation of the graphics process. Only programmer's who are considering enhancements or additions to the graphics process need consider reading this manual.

1.2 Where to Find Zebra Documentation

The document distribution is available as a separate tar file, with a name such as the following: 

Zebra-4.2.doc.tar.gz

See the Installation Manual for details on obtaining the document distribution.

All of the Zebra documentation and online help topics, as well as articles, tutorials, and presentations, are available through the Zebra home page: 

http://www.atd.ucar.edu/rdp/zebra.html

The home page provides access to both the PostScript and HTML formats of the manuals. The hypertext files are also included in the document distribution, as noted in Section 1.4, "Hypertext".

1.3 Hardcopy

The PostScript versions of the Zebra manuals can be found in the zebra/doc/ps directory of the Zebra document distribution. To print any of the manuals, first change to that directory. Then you can send the document file directly to a PostScript printer with the appropriate UNIX print command.

For example, use the lpr command under SunOS 4.x and other BSD systems: 

lpr -Pname file

Under Solaris, HP-UX, and other System V systems, use 


lp -dname file

Where name is the name of the PostScript printer, and file is the name of the manual file to be printed. The README file in the zebra/doc/ps directory lists the files along with the name and size of the document each file contains.

1.4 Hypertext

All of the Zebra documentation, except for the Graphics Process Internals manual, also exists in Hypertext Markup Language (HTML) format. The HTML files reside in the zebra/doc/html directory. Each manual has its own subdirectory, such as AdvUsers or Install. The top-level HTML file in each manual directory is simply the subdirectory name with a `.html' extension, such as AdvUsers.html and Install.html. To browse a manual, use a Web browser's "Open local file" function or put the path to the HTML file on the command line: 

% netscape /zebra/doc/html/AdvUsers/AdvUsers.html

1.5 Related Documentation

The documents below cover Zebra or related software; some may be helpful to anyone wanting further information about the third-party tools utilized by Zebra. In particular, the Unidata netCDF library has gained much popularity, and many applications in the scientific community support it.

  1. Corbet, J. M., Mueller, C., Burghart, C., Gould, K., and Granger, G., 1994: "Zeb: Software for Integration, Display, and Management of Diverse Environmental Datasets", Bulletin of the American Meteorological Society, 75:5, May, 1994, pp 783-792.

  2. Corbet, J. M. and Mueller, C., 1991: "Zeb: Software for Data Integration, Display, and Analysis", in Preprints of the 25th Conference on Radar Meteorology, American Meteorological Society, pp. 216-219.

  3. Jenter, H. L. and R. P. Signell, 1992. "NetCDF: A Freely-Available Software-Solution to Data-Access Problems for Numerical Modelers". Proceedings of the American Society of Civil Engineers Conference on Estuarine and Coastal Modeling. Tampa, Florida. See the file unidata.ucar.edu:/pub/netcdf/utilities.txt for availability.

  4. Rew, Russ; Davis, Glenn; and Emmerson, Steve; 1993. "NetCDF User's Guide: An Interface for Data Access, Version 2.3". Unidata Program Center, University Corporation for Atmospheric Research, Boulder, Colorado. Included in the netCDF distribution. See also the Unidata WWW page: 



http://www.unidata.ucar.edu/packages/netcdf


  1.6  On-line Help





Zebra supports an on-line help system which can display information on many parts of Zebra while the system is running. See "Display Interface Basics" on page 25.

  1.7  World Wide Web Tutorial





A hypertext Zebra tutorial begins at this URL:

http://www.atd.ucar.edu/rdp/zebra/tutorial/intro.html



Any WWW browser capable of inline images will be sufficient for showing the tutorial. Note that this tutorial is also accessible through the Zebra home page, mentioned below in Section 1.9.

The tutorial is most useful if followed while simultaneously running the Zebra COARE project demo. Also see the following chapters, especially "Display Interface Basics" on page 25, for basic introductions to starting and using Zebra which will augment the tutorial.

The original version of this manual, called "Getting Started", contained a tutorial in one of the chapters. That tutorial has been removed in favor of the more complete and updated hypertext version above. If you do not have a WWW browser, see the Zebra Installation Manual for information on obtaining a publicly available one. If you also do not have Internet access, contact one of the RDP support persons and we may be able to send you your own copy of the tutorial for browsing on your local machine.


  1.8  Mailing List





If your questions, comments, or concerns are of general interest to the Zebra user community, you can send email to the Zebra users' mailing list at this address:

zebra-users@atd.ucar.edu



Mail sent to this address will go to everyone on the mailing list. If you are not on the list and wish to be added, or if you wish to be taken off of the list, email to this address instead:

zebra-users-request@atd.ucar.edu



Mail sent to this address only goes to the administrator of the list. Please do not send requests for addition or removal to the general list, only to zebrausersrequest.

  1.9  Internet Assistance





All of the Zebra manuals and help topics can be browsed through the Atmospheric Technology Division's (ATD) World Wide Web server. The home page for Zebra announcements, tutorials, papers, manuals, and help topics can be found at this URL:

http://www.atd.ucar.edu/rdp/zebra.html



For assistance with the use or installation of Zebra, or to send bug reports, please email this address:

rdp-support@stout.atd.ucar.edu
