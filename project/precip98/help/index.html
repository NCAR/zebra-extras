CASES Zebra Project

Welcome to the CASES Zebra configuration. These help pages describe the particular capabilities of this CASES project. For general information about Zebra, see the help topics available on the Zebra icon bar, or visit the Zebra home page.

Printing Zebra Images

If you are logged into zebu as user display, you can make a color print of a Zebra plot window with this command:

% zebp
